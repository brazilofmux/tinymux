# UI Systems

## Hotkey System

### HotkeyEvent (extends BaseEvent)
Represents a keyboard binding with event actions:
- `PTKeyCombo *_rdKeyCombo` — key code + modifier mask
- `PTHotKey *_rdKey` — system registration object
- `BOOL _rdGlobal` — global vs. window-scoped
- `registerKeyBinding` / `unregisterKeyBinding` — system registration
- `execute:` — fires associated actions

### HotkeyCollection (extends EventCollection)
Container of all hotkeys:
- `executeForKey:modifiers:` — dispatcher when key pressed
- Matches against registered PTKeyCombo objects

### PTKeyCombo / PTHotKey
From PeterTonner framework:
- PTKeyCombo: key code + modifier mask pair
- PTHotKey: Carbon/Cocoa hotkey registration

### HotkeyPreferences
Preference pane UI for hotkey management:
- Displays list with EventEditor integration
- Add/edit/delete hotkey bindings

## Toolbar System

### ToolbarOption Protocol
```objc
- (BOOL)validForState:(AtlantisState *)state   // Is button enabled?
- (void)activateWithState:(AtlantisState *)state // Button clicked
- (NSString *)toolbarItemIdentifier              // Unique ID
- (NSToolbarItem *)toolbarItemForState:          // NSToolbarItem to display
```

### ToolbarUserEvent (extends BaseEvent, adopts ToolbarOption)
User-created toolbar buttons:
- `_rdToolbarIcon` — button image
- `_rdUUID` — unique identifier
- Inherits actions from BaseEvent (fires when clicked)

### Built-in Toolbar Items
| Class | Purpose |
|-------|---------|
| ToolbarConnection | Connect/disconnect toggle |
| ToolbarSearch | Search in scrollback |
| ToolbarAddressBook | Open address book |

### ToolbarCollection (extends EventCollection)
- `identifiers` — all visible button IDs
- `eventForIdentifier:` — get event for ID
- Serializes ToolbarUserEvent instances for persistence

## Preferences System

### AtlantisPreferencePane Protocol
```objc
- (NSString *)preferencePaneName     // Display name
- (NSImage *)preferencePaneImage     // Toolbar icon
- (NSView *)preferencePaneView       // Content view (from XIB)
- (void)preferencePaneCommit         // Save on OK
```

### AtlantisPreferenceController
NSWindowController + NSToolbarDelegate:
- `_rdPreferencePanes` — dict keyed by name
- `_rdPreferenceOrder` — display order
- Toolbar buttons select pane; clicking swaps content view

### Built-in Preference Panes
| Pane | Purpose |
|------|---------|
| GeneralPreferences | App-wide behavior |
| NetworkPreferences | Timeouts, encoding, TLS |
| WindowingPreferences | Window behavior, layouts |
| WorldDefaults | World-specific defaults |
| HotkeyPreferences | Hotkey editor |

## Log System

### RDLogType (abstract base)
```objc
- (id)initWithFilename:forSpawn:inWorld:[withOptions:]
- (void)openFile / closeFile
- (BOOL)isOpen
- (void)writeString:withState:    // Main append (with state for filtering)
- (BOOL)shouldWriteForState:      // Filter check
+ (NSString *)logtypeName
+ (BOOL)canAppendToLog
+ (BOOL)supportsOptions
- (NSView *)configureOptions:     // Format-specific UI
```

### Log Formats
| Class | Format | Notes |
|-------|--------|-------|
| RDPlainText | Plain text | One line per entry, minimal |
| RDFormattedText | Rich text | Word wrap, indentation, line skip/condense |
| RDHTMLLog | HTML | CSS support, header/footer, title, nbsp |

### Log UI
- **RDLogOpener**: Save panel to create new log
  - Format selector (popup button)
  - Options: spawn-only, include scrollback, timestamps
  - Delegates to RDLogType for format-specific config
- **RDLogCloser**: Panel to select and close active logs

### Integration
World instance holds array of RDLogType objects. When text arrives
at spawn, world calls each log's `writeString:withState:` which
filters and appends.

## Upload System

### UploadEngine (abstract base)
Timer-driven file upload:
```objc
- (id)initWithFile:forWorld:atInterval:withPrefix:suffix:
- (void)fireTimer      // Read chunk, send to world
- (void)finishUpload   // Cleanup on complete
+ (NSString *)uploaderName
```

### Upload Types
| Class | Purpose |
|-------|---------|
| CodeUploader | Upload code files (LPC/MUSHcode) with prefix/suffix |
| TextfileUploader | Upload text files with wrapping |

### UploadPanel
NSPanel UI for file selection, progress display.

## Misc UI

### Text Input
- **RDMUSHTextView**: Extended NSTextView for MUD input
  - Command history, cursor management
- **RDAtlantisTextInputView**: Alternative embeddable input
- **MUSHTextEditor**: Standalone editor window

### Progress & Status
- **AtlantisProgressSheet**: Modal progress with title, message, bar
- **AtlantisAboutBox**: Credits and version info
- **RenameWindowPanel**: Dialog to rename windows/spawns

### Pattern Matching
- **RDStringPattern**: OgreKit regex + simple matching
  - Types: BeginsWith, Is, Contains, Regexp
  - `patternMatchesString:`, `stringContainsPattern:`
  - NSCoding for persistence
